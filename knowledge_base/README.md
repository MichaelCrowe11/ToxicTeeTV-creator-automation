# ToxicTeeTV Knowledge Base

This folder contains normalized datasets and playbooks to operate the brand.

## Files

- `brand_profile.yaml`: canonical identity, voice, and positioning
- `platform_registry.csv`: account map across channels
- `cam_menu_dataset.csv`: digital offer catalog and pricing map
- `content_ideas_dataset.csv`: content idea bank with TikTok-safe translation
- `comment_response_playbook.csv`: reusable response templates
- `tiktok_safe_content_system.md`: TikTok-safe operating rules
- `compliance_boundaries.md`: boundaries and moderation constraints

## Security

- Do not store passwords or full credentials in this repository.
- Keep secrets in local `.env` (ignored by git) or a password manager.
