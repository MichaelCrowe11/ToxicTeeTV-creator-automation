# ToxicTeeTV Creator Automation

Operational system for ToxicTeeTV growth and monetization on creator platforms.

## Included

- 30/60/90 execution boards
- Content selection and scoring system
- Source inventory from initial album batch
- Initial 14-day implementation queue

## Quick Start

1. Review the board in `ToxicTeeTV_30-60-90_execution_board_from_album_1BJcPXPK3w1ipQ947.md`.
2. Execute Day 1 tasks in `implementation/day_1_7_task_pack.md`.
3. Run the first two weeks from `implementation/first_14_day_publish_queue.csv`.
4. Track outcomes and update priorities weekly.

## Structure

- `implementation/`: immediate execution artifacts
- `docs/`: operational notes
- `scripts/`: helper utilities for queue generation and scoring
